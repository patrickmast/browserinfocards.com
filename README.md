# BrowserInfoCards

A web app that displays browser and system information in a beautiful card-based UI.

**Live**: https://www.browserinfocards.com

## Development

```bash
bun install
bun run dev
```

## Deployment

```bash
./§deploy
```

This will:
1. Increment build version
2. Build the app
3. Deploy to Server1
4. Fix file permissions

## Tech Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS
- Wouter (routing)
- TanStack Query
